#
## RPM
> Red hat Package Manage
### 查询 `rpm`
#### `-qa |grep packagename`
#### `-q packagename`
#### `-qi packagename`
* information
#### `-ql packagename`
* install file local
#### `-qf /dictionary`
* owner of this dictionary
### 卸载
#### `rpm -e packagename`
### 安装
#### `rpm -ivh packagefile`
* `-i` -- install
* `-v` -- verbose 提示
* `-h` -- hash 进度条

## YUM
#### `yum install packagename`
