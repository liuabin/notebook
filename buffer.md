# OS 概述

## OS 概念

> Operating System

* 对下：管理各种系统资源
* 对上：为用户提供友好的界面
  * GUI Graphic User Interface

## OS 历史

* 手动操作阶段
  * 穿孔纸带、光电输入机
  * 用户在作业处理中，独享系统中全部资源；手工操作时间长
* 批处理阶段 -- OS雏形
  * 联机批处理
    * IO 传输由处理器机完成
  * 脱机批处理
    * IO 由卫星机实现
* 执行系统阶段
  * 通道/IO处理机 -- 自己指令系统和运控部件
  * “假脱机”
* 多道批处理
  * 内存同时存放多道作业
* 分时系统
  * 脱机方式非常不利于程序的动态调试
  * 对话方式，为各终端服务
  * OS 趋于完善
* 实时处理系统
* 通用操作系统
