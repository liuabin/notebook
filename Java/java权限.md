# Java 权限

> 指明可用性
* 类库开发人员
* 客户端程序员
> 包内单一名字空间

* Java源代码文件
  * 编译单元
* 主类
  * 构件 --> 包
* 支持类
  * 包外不可见

---

## default

* 包访问权限、类群聚
* 访问器 *accessor*
* 变异器 *mutator*

## public

* 接口访问权限

## private

* 本类访问
* getter
* setter

## protected

* 继承访问权限
* 子类可访问
* （可跨包）
