# Git

## General

* 分布式版本控制系统;每人电脑都是正确的版本
* 集中式版本控制系统(CVS，SVN);版本库是集中存放在中央服务器的

## History

* 02年，BitKeeper提供代码托管服务
* 05年，BitKeeper由于侵权收回
* Linus花了两周时间自己用C写了一个分布式版本控制系统，这就是Git
* 08年，GitHub上线
